## 🎨 Color Scheme & Visual Elements

### Colors Used
```
Primary Gradient:
  - Top: #667eea (Indigo)
  - Middle: #764ba2 (Purple)
  - Bottom: #ec4899 (Pink)

UI Elements:
  - Primary Button: #6366f1 (Indigo)

  - Danger Color: #ef4444 (Red)

Glass Effect:
  - Background: rgba(255, 255, 255, 0.1)
  - Backdrop Filter: blur(10px)
```

```
Heading: Bold, 24-48px
Body: Regular, 14-16px
Mono (Numbers): Courier, 14px
```

```
Card Entrance: scale(0.9) → scale(1), fade in
```

## 🎬 User Journey Visualization

```
Start App
  ↓
[2.5s Loading] ─→ Database Init Success
  ↓
Show Auth Modal
  ├─→ [Guest] ─────→ Skip to Game
  ├─→ [Email] ─────→ Register/Login
  └─→ Either way...
  ↓
Resolve Puzzle
  ├─→ Click Cell (Sudoku) ─→ Enter Number ─→ Cell Updates
  ├─→ Toggle Cell (Pattern) ──→ Color Changes
  ├─→ Enter Number (Sequence) → Input Validation
  └─→ All Options...
  ↓
Timer Running ⏱️ (increments every second)
  ↓
Use Hints (Optional)
  ├─→ 3 hints available
  ├─→ Click button
  └─→ Counter decreases
  ↓
Submit Solution
  ├─→ Validate locally (no server)
  ├─→ Show result (success/error)
  ├─→ If success:
  │   ├─→ Play success animation
  │   ├─→ Update points
  │   ├─→ Update streak
  │   └─→ Show menu
  └─→ If error: Allow retry
  ↓
Save Progress to IndexedDB
  ↓
Ready for Tomorrow's Puzzle!
```

| Effect | Trigger | Animation |

| Number Button | Mouse Hover | Scale + brightness |
