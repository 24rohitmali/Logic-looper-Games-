## 🔧 If Something Breaks

### Frontend won't load
```bash
cd frontend
npm install
npm run dev
```

### Backend won't start
```bash
cd backend
npm install
npm run dev
```

### Database connection error
```
Not critical! App works without database.
Frontend stores everything locally.
```

### Port already in use
```
Change port in:
- frontend: vite.config.js (port: 5174)
- backend: .env (PORT=3001)
```
